# Security Policy

## Scope

Colophon is a single-user tool meant to run on a trusted local network. It is not
hardened for exposure to the public internet, and running it there is outside its
intended use. Please keep this in mind when assessing risk.

## Supported versions

Security fixes are applied to the latest release. There is no long-term support
for older versions.

| Version | Supported |
|---------|-----------|
| 0.1.x   | yes       |

## Reporting a vulnerability

Please do not open a public issue for a security problem.

Report it privately using GitHub's [private vulnerability
reporting](https://github.com/LoneOutpost/colophon/security/advisories/new), or by
email to alexi@alexi.org.

Include the affected version, a description of the issue, and the steps to
reproduce it. You can expect an acknowledgement within a few days. Once the issue
is confirmed and a fix is available, the details will be disclosed and credit
given to the reporter unless you prefer to remain anonymous.
